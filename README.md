# Simple Visualisation of the Cocktail DB

Simple Visualisation of the Cocktail DB: https://www.thecocktaildb.com/ with ReactJS/GatsbyJS and TailwindCSS

Demo page: https://cocktails.lautenschlager.de


### Installing / Starting

```
npm i
npm start
```
